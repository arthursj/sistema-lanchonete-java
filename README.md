## üìò About

Hello! üëã
This repository was started with the goal of **consolidating and demonstrating my knowledge of pure Java**, applying **Object-Oriented Programming (OOP)** to simulate a **simple snack bar system running via console**.

The main goal of this project is to show, in a clear and practical way, that I understand:

* Creating and organizing classes in Java
* Fundamental concepts of **Object-Oriented Programming**

  * Classes and objects
  * Attributes and methods
  * Encapsulation (`private`, getters e setters)
* Using the **constructor** for object initialization.
* Using the keyword `this` to reference the current object.
* Creating and manipulating collections using `ArrayList`
* Implementing business rules within classes.
* Creation of methods for:

  * Add items to cart
  * Remove items from cart by code
  * Calculate the total purchase amount
* Using loops to iterate through collections
* Separation of responsibilities between classes (`Item`, `Carrinho` e `App`)
* Execution of the complete application flow using the method `main`
* Display results directly in the **console**.

This project **does not focus on graphical interfaces, databases, or frameworks**, but rather on **the fundamentals of the Java language, programming logic, and OOP**, simulating a common business scenario in a simple and didactic way.

The final result can be monitored directly in the **console**, demonstrating how the snack bar system works, from creating items to calculating the total purchase amount.
